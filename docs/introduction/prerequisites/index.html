Prerequisites

The following prerequisites apply for using Lumos to develop CKB DApps:

  • Basic Skills

    • Basic knowledge on Nervos CKB. For more information about the basic knowledge of Nervos CKB, see Nervos CKB Basics.
    • Knowledge of CKB Data Model. For more information about the CKB data model, see Nervos CKB Reference and CKB Data Structure.
    • Programming skills of Node.js projects for desktop applications or server applications

  • Development Stacks

    • Operating System: All major platforms, including Linux, Windows and macOS.

    • JavaScript runtime environment: Node.js LTS Version (>=12)

    • JavaScript package manager: Yarn or npm

    • Development tools to build native addons: GCC and make

    • Application development frameworks

      For example:

      • Electron (the framework for building cross-platform desktop applications)
      • Express.js (the web framework for Node.js)
