Workflow

The DApp development generally follows the following basic steps:

  1. Prepare the basic skills:

    • The basic knowledge on Nervos CKB. For more information, see Nervos CKB Basics.

    • The knowledge of CKB Data Model. For more information, see Nervos CKB Reference and CKB Data Structure.

    • Programming skills of Node.js projects for desktop applications or server applications.

  2. Set up the development environment.

    For more information, see Set Up the Development Environment.

  3. Create CKB accounts for the development and testing.

    For more information, see Create Accounts.

  4. Initialize a project and add Lumos packages as dependencies to the project.

    For more information about how to add Lumos packages to a project, see Install Lumos Packages.

  5. Establish the connection with the CKB network by the setup of the config manager and the Lumos indexer.

    For more information about how to set up the config manager, see Set Up the Config Manger.

    For more information about how to set up the Lumos indexer, see Set Up the Lumos Indexer.

  6. Program the DApp functions by using Lumos to deal with user queries and transaction requests.

    For more information about the usage of Lumos, see the sections in the Guides and Examples.
