Set Up the Development Environment

This guide will help you get your system set up for building DApps with Lumos. If you already have everything installed, read the other guides and walk through the examples to learn the usage of Lumos.

Install Node#

NodeJs is the runtime environment that must be installed on the system before using Lumos. The following sections explain the easiest way to install the Long Term Supported (LTS) version of Nodejs on Ubuntu Linux 20.04, macOS, and Windows 10.

  • Ubuntu 20.04
  • macOS and Windows 10

Install Node.js with Apt by Using a NodeSource PPA:

curl -sL https://deb.nodesource.com/setup_14.x -o nodesource_setup.sh
sudo apt install nodejs
Download and install the LTS version that is Recommended For Most Users.

Install Yarn#

It is recommended to install Yarn through the NPM package manager, which comes bundled with Node.js when it is installed on the system.

To install Yarn through NPM:

npm install --global yarn

Install Dependencies for node-gyp#

Lumos depends on node-gyp that is a cross-platform command-line tool for compiling native addon modules for Node.js.

node-gyp has a few additional system requirements and dependencies that have different installation steps on different operating systems.

  • Ubuntu 20.04
  • Windows 10
  • macOS
The development dependencies for Ubuntu 20.04 LTS are as follows:
  • Python v3.6, v3.7, v3.8, or v3.9 (Ubuntu 20.04 and other versions of Debian Linux ship with Python 3 pre-installed)
  • make
  • A proper C/C++ compiler toolchain, like GCC

To install GCC and make on Ubuntu 20.04, run the following command as root or user with sudo privileges:

$ sudo apt update
$ sudo apt install build-essential
The development dependencies for Windows are as follows:
  • Python 2.7
  • Visual C++ Build Environment
Installation Options
  • Option 1: Install all the required tools and configurations using Microsoft's windows-build-tools by running npm install -g windows-build-tools -vs2015 from an elevated PowerShell (run as Administrator).

    Note: This command installs all the system dependencies without conflicting with any software already installed on the system. Depending on the build tools' version, the installation requires 3 to 8 gigabytes space to get all dependencies installed. It can take at least 30 minutes depending on the network connection.

  • Option 2: Install dependencies and configure the tools manually.
    • Install Visual C++ Build Environment: Visual Studio 2015 Build Tools (using "Visual C++ build tools" workload) and run npm config set msvs_version 2015 in a cmd terminal.
    • Install Python 2.7 (v3.x.x is not supported), and run npm config set python /path/to/python2.7.
The development dependencies for macOS are as follows:
  • Python v3.6, v3.7, v3.8, or v3.9
  • Xcode and Xcode command line tools
For more information, see the instructions of node-gyp.

Download the CKB Pre-built Installer Package#

The pre-built installer package contains the tools that are used for running a CKB node. The package also contains ckb-cli that is a command line tool that is useful during the development and testing process.

The following example download the version 0.39.0. All releases can be found on the CKB releases page.

$ cd
$ curl -LO https://github.com/nervosnetwork/ckb/releases/download/v0.39.0/ckb_v0.39.0_x86_64-unknown-linux-gnu.tar.gz
$ tar xzf ckb_v0.39.0_x86_64-unknown-linux-gnu.tar.gz
