Workflow

The basic steps for developing a CKB DApp are as follows:

  1. Prepare the prerequisite skills and development stacks listed in Prerequisites.
  2. Install a CKB Node.
  3. Install Lumos packages for a Node project.
  4. Set up the config manager.
  5. (Optional) Set up the SQL database.
  6. Start the Indexer.
  7. Create an account for CKB transactions.
  8. Program CKB tasks. The DApps built with Lumos mainly deal with user queries and transaction requests, such as transferring capacity, deposits, withdrawals, and other operations of user assets.
