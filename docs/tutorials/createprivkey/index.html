Create an Extended Private Key

The HD wallet manager supports to generate extended private keys (privateKey and chaincode). The privateKey can be used to create accounts on chain.

Prerequisites#

The following prerequisites apply for creating an account by using the HD wallet manager:

  • The HD package (@ckb-lumos/hd) is installed in the project.

Examples#

Create an extended private key.#

/mydapp/src/index.ts
import { mnemonic, ExtendedPrivateKey } from "@ckb-lumos/hd";
const m = mnemonic.generateMnemonic();
const seed = mnemonic.mnemonicToSeedSync(m);
const extendedPrivateKey = ExtendedPrivateKey.fromSeed(seed);
console.log(extendedPrivateKey);

Result

ExtendedPrivateKey {
privateKey: '0x2c747d2b7a1e9d644f643be9e20e5c68d9ca5896772058e10605da5702b25b81',
chainCode: '0xe03a1dbfead6f021d67b4dfcc89c54884e5cded6c681cfe3a714a8b6a041b876'
}

The private key can be imported by using ckb-cli to create a new account.

For example:

$ echo 0x2c747d2b7a1e9d644f643be9e20e5c68d9ca5896772058e10605da5702b25b81 > pk
$ export TOP=$(pwd)
$ export PATH=$PATH:$TOP/ckb_v0.39.0_x86_64-unknown-linux-gnu
$ ckb-cli account import --privkey-path pk
